# QuizWiz — Stitch 设计说明（UI 生成用）

> **用途**：将本文作为 Stitch / AI UI 工具的输入，生成小程序与 Web 管理后台的界面。  
> **产品**：QuizWiz — 题库练习 + 班级学情 + 题目反馈的教培场景应用。  
> **语言**：界面文案为**简体中文**。风格：**干净、教育向、易读**，浅色背景为主，主色可用蓝或绿（与微信生态协调）。

---

## 全局约定

- **学生端**：微信小程序，页面路径见各节「路由」。
- **管理端**：浏览器 Web，路径前缀 `/admin-ui/`。
- **状态反馈**：加载用 skeleton 或 loading；错误用非阻断 toast + 简短说明；空列表有插图或占位文案。

---

## A. 微信小程序 — 学生端

### A1. 登录页

- **路由**：`pages/login/login`
- **目的**：微信授权登录后进入练习首页。
- **布局**：垂直居中卡片；标题「QuizWiz」或产品名；主按钮「微信登录」；底部小字说明失败时检查网络。
- **状态**：已登录则直接进入首页（无本页停留）。

---

### A2. 练习首页（核心）

- **路由**：`pages/index/index`
- **目的**：刷题、看统计、纠错反馈、收藏；可从班级作业带参数进入作业模式。
- **顶栏**：标题「练习」或「QuizWiz 练习模式」；右侧或第二行 **快捷入口**：练习历史、我的班级、错题本、收藏、开始练习（刷新题目）。
- **作业横幅**（条件显示）：黄/浅橙条 —「班级作业 · 请完成全部题目后提交」。
- **统计卡片**（白底圆角）：
  - 说明小字：统计按服务器日期，仅统计已提交练习。
  - 三列：**今日** / **近7日** / **累计**：做题数、正确数、正确率。
  - 可选折叠区：**按学科** 同上三组指标（有数据才显示）。
- **我的纠错反馈**（卡片）：
  - 标题「我的纠错反馈」+ 状态下拉（全部 / 已接收 / 处理中 / 已处理）+ 刷新。
  - 列表项：状态标签、原因类型、题目编号与题干摘要、处理备注（若有）。
- **筛选区**（作业模式下可隐藏或禁用部分项）：
  - 学科 picker、章节输入（可选）、出题方式（随机/顺序）、难度数字、题量滑块 1–20。
  - 错题入口场景：**仅重点复习** switch。
  - 提示：计时从首次选题起算。
- **题目列表**：每题为卡片 — 题干、题型（单选/多选）、选项列表（选中高亮）；题旁 **纠错** 文字链、**星标收藏**。
- **底部主按钮**：「提交并判分」（题目为空或提交中禁用）。
- **结果区**：得分、总用时、每题对错与解析、按钮「查看错题本」。
- **纠错弹窗**（模态）：标题「题目纠错」、原因 picker（答案可能有误 / 题干问题 / 选项问题 / 错别字 / 其他）、多行补充说明 ≤500 字、取消 / 提交。
- **空状态**：无题可练时的说明文案。

---

### A3. 练习历史

- **路由**：`pages/history/history`
- **目的**：查看历次练习会话与每题明细。
- **顶栏**：标题「练习历史」+ 刷新。
- **列表卡片**：时间、模式、学科、题量、得分与正确率、状态；按钮「查看详情 / 收起」。
- **展开区**：每题一行 — 对错样式区分（绿/红）、用户答案、正确答案、用时、解析；错题可 **加入/取消重点复习**。
- **空状态**：引导去练习。

---

### A4. 错题本

- **路由**：`pages/wrong/wrong`
- **目的**：按条件复习错题、标重点、标记已掌握。
- **顶栏**：标题「错题本」+ 「错题重练」。
- **筛选**：学科、章节、掌握状态（未掌握/已掌握/全部）、题量滑块、**仅重点复习** switch、「应用筛选」。
- **题目卡片**：题干、「重点」标签（若有）、学科/章节/错误次数/答案/解析；操作「标为重点」「标记已掌握」。
- **空状态**：鼓励文案。

---

### A5. 题目收藏

- **路由**：`pages/favorite/favorite`
- **目的**：浏览收藏并发起收藏练习。
- **顶栏**：「题目收藏」+「收藏练习」。
- **筛选**：学科、题量、「应用筛选」。
- **列表**：题干、元信息、「取消收藏」。
- **空状态**：引导在练习页点星标。

---

### A6. 我的班级

- **路由**：`pages/class/class`
- **目的**：邀请码入班、查看班级与班级作业。
- **区块1**：说明「向老师索取邀请码」；输入框；主按钮「加入班级」。
- **区块2**：「已加入的班级」— 班级名、加入时间；空态提示。
- **区块3**：「班级作业」— 每条：标题、班级名、题数、截止时间、逾期标签、已完成标签；右侧「去完成」或「再做一次」。

---

## B. Web 管理后台（教师 / 管理员）

**全局壳**：顶栏或侧栏导航 — 导入任务、题目反馈、班级学情、用户管理（仅管理员可见入口）、退出登录。

### B1. 登录页

- **路径**：`/admin-ui/login.html`
- **布局**：居中卡片 360px 宽；标题「管理后台登录」；用户名、密码；主按钮「登录」；灰色小字说明首次部署 bootstrap 账号。

---

### B2. 导入任务列表

- **路径**：`/admin-ui/index.html`
- **目的**：Excel 预检/正式导入、查看历史导入任务。
- **上传区**：文件选择（xlsx/xls）、预检、正式导入、下载模板；结果提示区。
- **筛选**：模式、状态、日期起止、查询、重置。
- **表格**：ID、模式、文件名、总行数、成功、失败、状态、更新时间、操作「查看详情」。

---

### B3. 导入任务详情

- **路径**：`/admin-ui/detail.html?id=`
- **目的**：查看每行导入结果与失败原因。
- **摘要卡片**：任务 ID、总行、成功行、失败行。
- **失败原因统计**（列表）。
- **工具栏**：仅失败行 checkbox、刷新、导出 CSV。
- **表格**：行号、状态、原因、消息、原始数据 JSON。

---

### B4. 用户管理（管理员）

- **路径**：`/admin-ui/users.html`
- **创建区**：用户名、初始密码、角色（teacher/admin）、创建按钮。
- **查询区**：关键字、角色、启用状态、查询。
- **表格**：ID、用户名、角色、状态、创建时间；行内按钮：切换角色、启用/禁用、重置密码。

---

### B5. 班级学情（教师与管理员）

- **路径**：`/admin-ui/classes.html`
- **班级区**：输入新班级名、创建；下拉选择当前班级；删除当前；刷新。
- **邀请码区**（选中班级后）：大字号邀请码、复制按钮；说明发给学生在小程序输入。
- **统计窗口**：7 / 14 / 30 天。
- **看板**：若干指标卡片（练习量、正确率等 — 用网格排列）。
- **成员练习概况表**：学生 ID、答题次数、正确数、正确率、操作。
- **班级易错题表**：题目 ID、错题次数、总作答、错率、题干摘要。
- **班级作业**：
  - 题目搜索：关键词、章节、难度、题型、状态、搜索、分页、勾选表格、已选计数、清空、标题输入、截止时间 datetime、发布；可选 textarea 填题目 ID。
  - 作业列表表：标题、截止、题数、完成进度、操作。
- **成员管理**：输入学生 ID、备注、加入；成员表与移除。
- **学生答题时间线**（展开面板）：选中学生后展示时间线列表。

---

### B6. 题目反馈（教师与管理员）

- **路径**：`/admin-ui/reports.html`
- **顶部仪表盘**：反馈总量、待处理、处理中、已关闭、平均处理时长、SLA 超时数；下方趋势/学科摘要一行文字。
- **筛选栏**：状态、视图（明细/按题聚合）、排序、仅高风险、仅 SLA 超时、超时阈值输入、批量按钮（处理中、关闭、备注、待修复）、刷新。
- **明细表**（多选列）：复选、ID、时间、学生 ID、题目 ID、题干摘要、类型、说明、状态、待处理时长（超时标红标签）。
- **按题聚合表**：题目 ID、优先级徽章、最久待处理、近 7 天次数、最近反馈时间、反馈总数、状态分布、题干摘要、操作（管理员：修复上架、替换题目、删除题目等 — 危险操作用红色按钮）。

---

## C. 设计令牌建议（可调整）

- **圆角**：卡片 8–12px；按钮 8px。
- **字号**：标题 18–20；正文 14–15；辅助 12–13。
- **颜色**：背景 `#f5f5f5`；卡片白；主色 `#1677ff` 或教育绿 `#07c160`；错误 `#cf1322`；警告/逾期 `#ad6800`。
- **小程序**：遵循微信导航栏规范；底部安全区留白。
